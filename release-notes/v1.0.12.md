## Modifiche ai File

### backend.py
- Aggiunta funzione `log_with_timestamp()` per logging con timestamp millisecondi
- Aggiunta funzione `wait_for_selector_with_retry()` con:
  - Retry logic con max 3 tentativi (configurabile)
  - Exponential backoff tra i tentativi
  - Verifica stato 'networkidle' prima di cercare elementi
  - Screenshot automatici su fallimento (salvati in logs/)
  - Logging dettagliato di ogni tentativo
- Aggiornata funzione `wait_for_loading_spinner_to_disappear()` per accettare timeout configurabile
- Caricamento parametri timeout da config.ini
- Applicato retry logic al selettore critico #codiceTariffa

### config.ini
Nuova sezione parametri timeout:
- `TIMEOUT_DEFAULT = 120000` (2 minuti per operazioni standard)
- `TIMEOUT_SELETTORE_CRITICO = 180000` (3 minuti per selettori critici)
- `MAX_RETRY = 3` (numero massimo di tentativi)
- `WAIT_AFTER_SPINNER = 2` (delay in secondi dopo scomparsa spinner)

### .gitignore
- Aggiunta cartella `/logs/` per escludere screenshot di debug

### package.json
- Versione aggiornata da 1.0.11 a 1.0.12
